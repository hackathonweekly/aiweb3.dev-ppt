代码提交指南

GitHub 代码提交流程

1

Fork官方仓库

访问 github.com/openbuildxyz/ai-web3-growth-journey 并点击 Fork

2

克隆Fork后的仓库

git clone https://github.com/你的用户名/ai-web3-growth-journey.git
cd ai-web3-growth-journey
3

创建新分支

git checkout -b 团队名-project
4

添加项目文件

在 projects/2025-03/团队名/ 目录下添加项目文件

5

提交更改并创建PR

git add .
git commit -m "Add 团队名 project"
git push origin 团队名-project

然后创建Pull Request

必须提交的文件

  • README.md: 项目介绍、团队信息、技术实现
  • contracts/: 智能合约代码
  • frontend/ (可选): 前端代码
  • frontend/ (可选): PPT，参考教程
  • demo.md: 包含Demo链接和截图
  • prompts.md: 记录与AI助手的关键提示词交互

提交要求

  • 代码必须包含详细注释，特别说明AI生成部分
  • 记录与AI的关键交互提示词，展示AI如何帮助完成项目
  • 必须提交完整的README.md说明
  • 创建展示页面，包含项目概述与技术实现
